# 本项目部署方式

本项目通过 vercel 自动部署，暂时并未使用自己的域名。目前可以通过 yucohny.vercel.app 访问。

# 本项目参与方式

如果您想修改界面样式、添加新的笔记篇章，或直接加入项目编辑，请直接联系邮箱 yucohny@163.com。

如果您想编辑（添加内容，修改内容）已有篇章，可以将项目 fork 后创建新的分支，发布新的分支后提新的 pr，我会在第一时间尽快 review 以及合并。

请注意每一个新的 pr 仅修改一篇笔记，并且自动检测无冲突。

请保持 pr 命名格式：笔记所属大类缩写 + 具体笔记文件内容。

比如您想修改「计算机网络-跨站脚本攻击」一节，请将 pr 命名为：`cn-xss`。

比如您想修改「React-代码分割」一节，请将 pr 命名为：`react-codeSplitting`。
